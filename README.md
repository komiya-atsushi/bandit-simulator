bandit-simulator
================

Web-based bandit algorithm simulator with some front-end technologies (nvd3.js + d3.js + AngularJS).

Demo page
---------

See http://playground.k11i.biz/bandit-simulator/


Supported algorithms
--------------------

- ε-Greedy
- ε-Greedy w/ annealing
- Softmax
- Softmax w/ annealing
- UCB1
- Thompson sampling
